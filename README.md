# PortifolioDigital

Portfólio digital profissional de Murilo Henrique Quitzau de Oliveira, com foco em tecnologia, automação de processos, análise de dados e desenvolvimento de soluções corporativas.

## Objetivo

Este projeto transforma um currículo tradicional em uma apresentação interativa, moderna e pública para processos seletivos internos e externos.

## Foco do conteúdo

- Sobre profissional
- Tecnologias e nível de atuação
- Projetos em formato de estudo de caso
- Habilidades técnicas e comportamentais
- Experiência profissional em formato público
- Contato

## Privacidade e confidencialidade

Este portfólio segue uma versão pública:

- Dados corporativos sensíveis são removidos ou anonimizados
- Nomes internos, números estratégicos e contextos confidenciais não são expostos
- Exemplos e descrições podem ser adaptados para preservar sigilo empresarial

## Estrutura

- [index.html](index.html): página única com HTML, CSS e JavaScript

## Execução local

1. Clone o repositório.
2. Abra a pasta no VS Code.
3. Abra o arquivo [index.html](index.html) no navegador.

## Publicação no GitHub Pages

URL esperada:

- https://klebermurillo.github.io/PortifolioDigital/

Para publicar mudanças:

1. git add .
2. git commit -m "atualiza portfolio"
3. git push origin main

Após o push, aguarde alguns segundos e valide a URL do GitHub Pages.
